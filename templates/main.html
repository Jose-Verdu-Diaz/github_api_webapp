Buscador de repositorios

GitHub API webapp

Introduzca en el buscador un nombre de usuario. Los usuarios que contengan el nombre introducido aparecerán a continuación. Seleccione un usuario para ver una tabla de repositorios con información básica. Téngase en cuenta que si se realiza una búsqueda demasiado genérica y/o el usuario encontrado tiene una gran cantidad de repositorios se puede superar la tasa límite de consulta de la API.
